# -How-good-is-this-medical-biomarker-Project
The repository contains code used in an undergraduate project, titled "How good is this medical biomarker?", to apply the considered ROC induced-regression methodology to the Alzheimer’s Disease Neuroimaging Initiative (ADNI) dataset. Given access to the ADNI data, which is not publicly available (more information available at https://adni.loni.usc.edu/data-samples/adni-data/), the included files can be used to estimate covariate-specific ROC surfaces, and plots illustrating effects of covariates on estimated volume under the ROC surface and optimal thresholds based on the Youden Index and Minimum Euclidean Distance measures along with nonparametric bootstrap confidence intervals. Moreover, parts of the code used to carry out preliminary data analysis are included.
